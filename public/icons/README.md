# PWA Icons Needed

You need to create these icon files in the /public/icons/ folder:

- icon-16.png (16x16)
- icon-32.png (32x32)  
- icon-192.png (192x192)
- icon-512.png (512x512)

## Quick way to create them:

1. Go to https://favicon.io/favicon-generator/
2. Enter "O" as the text
3. Set background to #0a0a0a (black)
4. Set font color to #ea580c (orange)
5. Download and extract the icons
6. Rename them to match the names above

Or use any image editor to create a simple "O." logo in orange on black background.

The app will work without icons, but won't look right when installed.
